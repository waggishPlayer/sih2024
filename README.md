# AAYUSHVANA (SIH 2024)

AAYUSHVANA is a static frontend prototype that presents medicinal plant information, focused on plant details, disease treatment context, and a simple multi-page website flow.

## Project Overview

This repository contains a pure HTML/CSS website with:
- A main landing/details page (`index.html`) for a sample medicinal plant (Neem)
- Placeholder pages for Blogs and About sections (`blogs.html`, `about.html`)
- Static visual assets (JPG/PNG/SVG) used by the UI

The project appears to be an SIH 2024 submission/prototype emphasizing UI and content presentation.

## Tech Stack

- HTML5
- CSS3
- Static assets (images/icons)

No backend, build system, package manager, or test framework is configured in this repository.

## Pages

- `index.html`  
  Main UI with navbar, plant details (Neem), treatment information, recommendation cards, and footer/contact section.

- `about.html` and `blogs.html`  
  Currently “Under Construction” pages sharing `blogs.css` styling.

- `i.html`  
  An additional standalone prototype page with embedded styles.

## How to Run

Since this is a static website:

1. Clone/download the repository.
2. Open `index.html` directly in a browser.

For better local navigation behavior, you can also serve it with a simple local static server (optional).
